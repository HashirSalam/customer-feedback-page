<!-- GETTING STARTED -->

### Installation

1. Clone the repo

2. Go to the repo
3. Install NPM packages
   ```sh
   yarn install
   ```
4. Start the project
   ```sh
   yarn start
   ```

## Usage

Feel free to fill the form and the graph will show live stats according to the ratings.

3 ,2 stars -- > Good
